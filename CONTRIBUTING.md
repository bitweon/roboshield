# Contributing to RoboShield

## Development Process

1. Fork the repository
2. Create a feature branch

git checkout -b feature/my-feature

3. Commit changes

git commit -m "Add ROS2 audit check"

4. Push branch

git push origin feature/my-feature

5. Open a Pull Request

## Coding Standards

- Bash scripts must pass ShellCheck
- Include comments
- Test before submitting

## Security

Do not submit:
- Malicious code
- Hardcoded passwords
- Secrets or API keys
